Image Automation Pipeline

AI-powered dual ingest into ArchivesSpace and DSpace with bidirectional cross-linking

1

AI Image Captioning

Each image in the staging_images/ folder is sent to a locally-running AI vision model (LM Studio). The model examines each photograph and generates a title, description, and keywords. Results are saved to captions.csv.

caption_folder.py LM Studio (localhost:1234) Output: captions.csv
2

Dual-System Import

Reads captions.csv and creates records in both systems via their REST APIs. In DSpace, a community, collection, and items are created with full Dublin Core metadata and uploaded image bitstreams. In ArchivesSpace, Digital Objects are created with embedded images, summary notes, and keywords. Bidirectional cross-links connect corresponding records across both systems.

import_to_both.py DSpace REST API (:8081) ArchivesSpace API (:8089) --prefix flag for labeling
3

Thumbnail Generation

DSpace's built-in media filter processes uploaded bitstreams and generates thumbnail images, making them visible in browse and search interfaces.

dspace filter-media Runs inside Docker container
